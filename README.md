# book_review_analysis
## 데이터 수집
1. 도서 정보나루 사이트의 오픈 API 활용
   * 조건을 조합해서 여러 카테고리 별 그룹 만들기
     ```params = {
      "startDt": "2018-01-01",
      "endDt": "2018-12-31",
      "gender": '0',
      "age" : '0',
      "kdc100" : '0',
      "format" : 'json}
     ```
   * 그룹 별 인기 대출 순위에 따른 도서 목록 받기
    
2. 위에서 얻은 그룹 별 개별 도서를 '교보 문고' 웹사이트에 겁색 후, '책 소개' 웹 스크래핑
